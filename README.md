## Iluminati Project

Plataforma de vendas focada em segurança e observabilidade, construída com arquitetura de microserviços usando NestJS. Projeto de teste e exploração das seguintes ferramentas de observabilidade:

| Ferramenta | Função |
|---|---|
| **OpenTelemetry Collector** | Centralização e roteamento de telemetria |
| **Jaeger** | Armazenamento e visualização de traces distribuídos |
| **Prometheus** | Coleta e armazenamento de métricas |
| **Loki** | Armazenamento de logs estruturados |
| **Grafana** | Painel unificado de visualização |

### Microservices

- **gateway** — Ponto de entrada da plataforma
- **catalogo** — Serviço de catálogo de produtos
